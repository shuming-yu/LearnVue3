檔名：reactive_ref.html
  • 請都使用 const
  • ref 可以用在純值
  • ref 比較好用, 且不會出錯的方法
  • reactive 可省去 .value

    reactive

  • Handler(控制器) : 監控值的變化 ( 渲染畫面 or 重新調整原始值 )
  • target : 值

    • ref

  • Handler(控制器) : 監控值的變化 ( 注意: 只有單純的純值時, 控制器是不存在的, 但在原型內有相同機制做運作 )
  • rawValue : 實際上放原始值的地方
  • value : 外層來存取所使用的
  • 寫入值, 它會把值再寫一份到 rawValue 裡面
  • 讀取值, 它會在 rawValue 裡面把值讀取出來
  • 好處 : 它的 value 不一定是 proxy 物件, 一樣可使用純值方式進行存取
